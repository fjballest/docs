Q(1): Run a command using qlfs

USAGE

        lf[14]: stat /zx/usr/web/sys/man/1/-?: no such file or directory

DESCRIPTION

Q runs a command using qlfs(1) to submit it to ql(1). It is inteded as a shell for Acme and other IDEs, and also as an interface to run Ql commands on remote machines using their (mounted) Ql file systems.

Q sends all its standard input to the command input before actually starting the command.

EXAMPLES

Run date: 

        ; Q date </dev/null
    Tue May 19 18:49:23 CEST 2015

Run date on nautilus: 

        ; zxfs -n tcp!nautilus!ql /n/nql
    ; Q -q /n/nql date </dev/null
    Tue May 19 18:58:19 CEST 2015
