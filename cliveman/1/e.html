E(1): stream and file editor

USAGE

        usage: e [-Dnw] cmd {file}
        -D: debug
        -n: do not write resulting files

DESCRIPTION

E is a file and stream editor similar to the Sam editor as found in Plan9. It edits standard input or the named files according to the cmd given in the first argument. All files edited are printed in standard output or, if -w is given, are updated. When using both -w and standard input, the file is printed on standard output.

Flag -n can be used to prevent writing the resulting files after the edition. This is intented to let the user control what is printed by writing print commands.

The command is written in the Sam command language, as described in the references. However, e does not have the q and B commands by intention, and has a new command P that is like p but writing always the end of line after each print.

EXAMPLES

Replace A with B in standard input 

        ; e ', x/A/c/B'

Delete all runs of xxx in file and write the result to standard output: 

        ; e ', x/xxx/d' file

Delete all runs of xxx in file and update file with the result: 

        ; e -w ', x/xxx/d' file
