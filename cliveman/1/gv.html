GV(1): select greps in reverse.

USAGE

        usage: gr [-Daeflmsvx] rexp [rexp]
        -D: debug
        -a: print just addresses
        -e: extend regexps to match all the text
        -f: print addresses for matches in full files (like sam)
        -l: print just the names of matching files
        -m: print just matching text
        -s: just status
        -v: invert match
        -x: print selections for further editing commands

DESCRIPTION

Gv is gr(1) with flags -xvef. It is useful in pipes after a call to gx(1) to filter the selections that do not contain text matching the given expression.
