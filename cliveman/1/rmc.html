RMC(1): remove files

USAGE

        usage: rmc [-Dfnv] {file}
        -D: debug
        -f: force ignore of errors
        -n: dry run; implies -v
        -v: report operations made

DESCRIPTION

Rmc removes files.

EXAMPLE

Remove everything (do not try this at home): 

        ; rmc /,

Remove all Ada source files from /src: 

        ; rmc '/src,~\.ad[bs]$'

Using the same example, print one lne per file removed but do not remove it: 

        ; rmc -n '/src,~\.ad[bs]$'
