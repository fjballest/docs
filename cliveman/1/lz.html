LZ(1): LIST ZX ATTRIBUTES

USAGE

    usage: lz [-Dadlprv] {file}
    	-D: debug
    	-a: list all attributes
    	-d: list dirs, not their contents
    	-l: long listing
    	-p: print just the name, not the full path
    	-r: recur through directories
    	-v: verbose listing, show all user attributes
    
    usage: l {file}
    

DESCRIPTION

Lz lists zx files and attributes on the name space (if defined) or the local
file system (if NS not defined).

The script l calls lz and columnates its output.

These flags are exclusive and determine how each entry is listed.
  -l:
        produce a long listing
  -v:
        produce a long listing with all user attributes
  -a:
        print all attributes

These other flags may be used in any case:
  -r:
        recur through directories.
  -d:
        list directories instead of their contents
  -p:
        print the file names instead of their paths

EXAMPLE

Produce a recursive listing:
    ; lz -lr
    /zx/sys/man/1
    /zx/sys/man/1/auth
    /zx/sys/man/1/chz
    /zx/sys/man/1/doc
    /zx/sys/man/1/hist
