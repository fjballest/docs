COMP(1): compare zx files

USAGE

        usage: comp [-Dl] file [file]
        -D: debug
        -l: long listing; print all the file with changes marked

DESCRIPTION

Comp compares files for differences, similar to the UNIX diff command. If a single file is indicated in the command line, standard input is compared with respect to it. If the indicated arguments select multiple files, they are compared one by one.

By default the program prints just the differences, use the flags to ask for more.

EXAMPLE

Compare go source files in the tree rooted at the current directory and the tree rooted at /tmp/x: 

        ; comp ',~\.go$' '/tmp/x,~\.go$'
    #diff    comp.go:208:    and    /tmp/x/comp.go:207:
    -         x.diff()
    #diff:    comp.go:209:    and    /tmp/x/comp.go:209:
    +     x.diff()
    ...

SOURCE

  • /zx/sys/src/clive/cmd/comp

BUGS

This tool was written for text files and does not handle binary files, but it should.
