Section 1: commands

  • intro: introduction to clive
  • auth: setup zx authentication
  • ch: change file attributes
  • cnt: count lines, words, runes, and bytes
  • cols: columnate words
  • diffs: compare zx files for differences
  • doc: clive manual documents
  • eco: echo
  • flds: print fields
  • frmt: format text files
  • go: go tool for the system
  • gp: grep files for predicates
  • gr: grep files
  • hist: dump history
  • importzx: import clive file trees
  • ix: ink shell and command executor
  • jn: join fields
  • lf: list files
  • lns: print lines
  • mvf: move files
  • on: notify the user or run commands when files or user status
  • path: path
  • pf: print files
  • ql: clive shell
  • rmf: remove files
  • rom: rom
  • runfor: run commands for input lines matching expressions
  • srt: sort lines
  • trex: translate expressions
  • web: serve web pages
  • wr: document writer
  • xcmd: execute commands
  • xp: evaluate expressions
  • xzx: export local files using the zx file protocol
  • zxdump: create a zx dump for file trees
  • zxfuse: import remote zx file trees through fuse
  • zxpull: pull changes for a replica
  • zxpush: push changes for a replica
  • zxrepl: create a file tree replica configuration
  • zxsync: synchronize a replica
