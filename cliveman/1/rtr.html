RTR(1): translate runes

USAGE

        usage: rtr [-Dltuv] [-d arg] [from to] {file}
        -D: debug
        -d: delete
        -l: convert to lowercase
        -t: convert to title
        -u: convert to uppercase
        -v: invert matches for deletes

DESCRIPTION

Rtr translates runes in files.

When given from and to it translates runes matching those in from to the ones specified in to. These arguments are sets and may contain runes (the same number of runes in each set), and/or rune ranges, specified as x-y. Both sets must describe the same number of runes to translate from one to another, or the to set may have a single rune to translate any rune in from to the rune in to.

Instead of this usage, the following flags are understood:

-u
translates runes to uppercase
-l
translates runes to lowercase
-t
translates runes as in a title case.
-d
deletes runes matching the argument set. If flag -v is given, matching is inverted.

EXAMPLES

Translate digits to letters starting at A: 

        ; echo A01Z | rtr 0-9 A-x
    AABZ

Make a title 

        ; echo lord of the rings | rtr -t
    Lord Of The Rings

Replace any digit with an X in the input: 

        ; echo ab012cd | rtr 0-9 X
    abXXXcd
