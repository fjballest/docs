SYSGO(1): go tool for the system

USAGE

        usage: sysgo go-options...
    usage: sysgo doc clive/...

DESCRIPTION

Sysgo sets the GOPATH and GOBIN variables for the standard clive locations and calls go(1) with the given arguments.
