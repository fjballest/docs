WF(1): write files

USAGE

        usage: wf [-DLSVadflnpsvwx] {file}
        -D: debug
        -L: longer than long list dirs
        -S: print strings enclosed in [], for debugging
        -V: very verbose list; all attributes
        -a: print last gr addresses
        -d: do not print file data
        -f: do not print dir data
        -l: long list dirs
        -n: dry run for -w, implies -w
        -p: print abs paths
        -s: print strings
        -v: verbose list dirs; or verbose report of files written back
        -w: write back input files (eg. after gr -x pipes)
        -x: print all gr addresses

DESCRIPTION

Wf is used to write back to disk files modified through a pipe. It is pf(1) with flag -w, see this page for examples.

SOURCE

SEE ALSO
