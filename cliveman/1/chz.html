CHZ(1): CHANGE ZX ATTRIBUTES

USAGE

    usage: chz [-D] attr value {file}
    	-D: debug
    	attr uids means all of Uid, Gid, Wuid
    	attr mode can use +rwx or -rwx as value; name optional here
    

DESCRIPTION

chz changes zx attributes for files on a local file system. It is most useful on
UNIX to change the attributes and uids as used by zx.

Only one attribute is changed at a time, however, the attribute uids can be used
to change all of Uid, Gid, and Wuid. The attribute mode admits as value the form
+rwx and the form -rwx to add or remove one of those.

EXAMPLE

Add read permission to all files within the current directory.
    ; chz mode +r ,
