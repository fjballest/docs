DIFFS(1): compare zx files for differences

USAGE

        usage: diffs [-Dlqu] file [file]
        -D: debug
        -l: long output
        -q: quiet output (report which files differ)
        -u: use unix out

DESCRIPTION

Diffs compares files for differences, similar to the UNIX diff command. One set of files to compare is given through the command input and the other as an argument. Or both are given as arguments.

EXAMPLE

Compare two trees rooted at /tmp/echo and ./echo and report the differences: 

        ; diffs -u /tmp/echo,- echo,-

Note that you might use more elaborate predicates, for example to select just Go source files.

SOURCE

  • /zx/sys/src/clive/cmd/diffs

BUGS

This tool was written for text files and does not handle binary files, but it should.
